Loops execute code **repeatedly** while a condition holds true.

### ðŸ” `while` Loop

```apex

```

### ðŸ” `do-while` Loop

Runs **at least once**, checks condition **after** execution.

```apex

    // Code to run

```

### ðŸ” `for` Loop

Used when number of iterations is **known**.

```apex

    // Code to run
}
```

### âœ… Example: Countdown Loop
